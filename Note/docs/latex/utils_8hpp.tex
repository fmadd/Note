Authenticates a user by sending authentication data securely over a TCP socket. 

This function authenticates a user by sending their login and encrypted password over the provided TCP socket. It first sends an authentication number to indicate the type of operation. The password is encrypted using AES encryption with the given AES key.


\begin{DoxyParams}{Parameters}

\hline

\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
Returns true if authentication succeeds, false otherwise. The function also sets the global variable {\ttfamily current\+\_\+user} to {\ttfamily login} upon successful authentication.
\end{DoxyReturn}
The function performs the following steps\+:
\begin{DoxyItemize}
\item Sends an authentication number (integer) to the server via the socket.
\item Encrypts the plain text password using AES encryption with {\ttfamily aes\+Key}.
\item Sends the length of the login and encrypted password as size\+\_\+t values followed by the actual login and password data over the socket.
\item Reads a boolean response from the socket indicating the success of the authentication.
\item If authentication is successful (response is true), updates the global {\ttfamily current\+\_\+user} variable.
\end{DoxyItemize}
